## 语言特性
#### 浮点数在计算机中存储的是近似值，float保存有精度缺失
### 小数的四舍五入（保留三位小数）：对第四位转化
#### a=floor(a*1000+0.5)/1000; 
#### //假如这个小数位的数值大于5，加上0.5能够实现进位的作用，倘若小于0.5，就没有实现进位的作用。如此，就能实现小数位的四舍五入。
#### //floor为double型函数，用于向下取整
#### //C语言的隐式转换规则：当 double 和 int 进行算术运算时，int 会自动提升为 double，所以 / 100 等价于 / 100.0，除法按双精度浮点执行
### 判断浮点数是否相等:判断两个浮点数的差是否小于一个很小的容忍误差（如 1e-9），(不能直接用==！)
#### if (fabs((a - b) - c) < 1e-9) 
#### //fabs(x):返回 x 的绝对值，返回值类型为 double (abs用于整数！）
